# ProdeskITMission3
Dev Detective is a modern GitHub user search app that fetches and displays profile details using the GitHub API. It shows user info, top repositories, and includes a Battle Mode to compare two developers based on followers. Built with HTML, CSS, and JavaScript using async/await.
